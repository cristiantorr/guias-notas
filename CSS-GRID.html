Tutorial CSS Grid - Explicación de cada propiedad

1. display: grid

La propiedad display: grid convierte el contenedor en un grid container, habilitando el uso de propiedades de grid en sus elementos hijos.

2. grid-template-columns

La propiedad grid-template-columns define cuántas columnas tendrá el grid y su tamaño. Usamos unidades como px, %, fr, etc.

3. grid-template-rows

La propiedad grid-template-rows define las filas del grid. Como en las columnas, puedes usar unidades como px, %, fr o auto.

El grid tiene dos filas: la primera con 150px de altura y la segunda con 200px.

4. gap

La propiedad gap define el espacio entre los elementos dentro del grid. Es útil para mantener el espacio entre filas y columnas.

5. grid-column

La propiedad grid-column especifica en qué columna comienza y termina un item. Esto permite que un item ocupe varias columnas.

6. grid-row

La propiedad grid-row funciona de manera similar a grid-column, pero para las filas. Define dónde empieza y termina un item en el eje vertical.

7. grid-template-areas

La propiedad grid-template-areas permite nombrar las áreas del grid. Hace que la disposición de los elementos sea más comprensible y visual.

Header
Sidebar
Content
Footer

En este ejemplo, hemos definido áreas: grid-template-areas: "header header header" "sidebar content content" "footer footer footer";. Los items se colocan automáticamente según el área que les asignamos.

8. grid-area

La propiedad grid-area se utiliza para colocar un item en una área específica del grid, previamente definida con grid-template-areas.

Sidebar

Usando grid-area, colocamos los items en las áreas definidas por grid-template-areas. Así, cada elemento se coloca en su área correspondiente como header, sidebar, etc.
