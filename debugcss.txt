  outline: 1px solid rgba(255, 0, 0, 0.5);
}
img {
  outline: 1px solid rgba(0, 255, 0, 0.5);
}
section {
  outline: 1px solid rgba(0, 0, 255, 0.5);
}


1. Debug de overflow (el más común)

Para encontrar qué elemento rompe el ancho:

* {
  outline: 1px solid red;
  max-width: 100vw;
}

2. Debug por niveles (colores por profundidad DOM)

Excelente para entender layouts complejos:

* { outline: 1px solid red; }
* * { outline: 1px solid green; }
* * * { outline: 1px solid blue; }
* * * * { outline: 1px solid orange; }

3. Debug de grid y flex (CLAVE)
Flexbox
* {
  display: flex !important;
  outline: 1px solid red;
}

Grid
* {
  outline: 1px solid red;
}

*::before {
  content: attr(class);
  font-size: 10px;
  color: red;
}

4. Debug de imágenes rotas / mal escaladas
img {
  background: rgba(0,255,0,.1);
  outline: 2px dashed lime;
}

5. Debug de spacing (padding y margin)
* {
  background-color: rgba(255,0,0,.05);
}

6. Debug de elementos invisibles
*[style*="display: none"],
*[style*="visibility: hidden"] {
  outline: 2px dashed red;
}

7. Debug de z-index (cuando algo no se ve)
* {
  position: relative;
  z-index: 1;
}

8. Debug de interacción (hover / focus)
*:hover {
  outline: 2px solid blue;
}

*:focus {
  outline: 2px solid orange;
}

9. Debug de accesibilidad (muy top)
button:not([aria-label]):not(:has(*)) {
  outline: 3px solid red;
}

10. Debug activable por clase (recomendado)
.debug * {
  outline: 1px solid rgba(255,0,0,.4);
}

.debug img {
  outline: 2px solid lime;
}

.debug section {
  outline: 2px solid blue;
}
