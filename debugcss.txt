1. Debug de overflow (el más común)

Para encontrar qué elemento rompe el ancho:

* {
  outline: 1px solid red;
  max-width: 100vw;
}

2. Debug por niveles (colores por profundidad DOM)

Excelente para entender layouts complejos:

* { outline: 1px solid red; }
* * { outline: 1px solid green; }
* * * { outline: 1px solid blue; }
* * * * { outline: 1px solid orange; }

3. Debug de grid y flex (CLAVE)
Flexbox
* {
  display: flex !important;
  outline: 1px solid red;
}

Grid
* {
  outline: 1px solid red;
}

*::before {
  content: attr(class);
  font-size: 10px;
  color: red;
}
